表格结构说明：

- **`ord`**: 范数的类型参数。
- **`norm for matrices`**: 对于矩阵使用的范数定义。
- **`norm for vectors`**: 对于向量使用的范数定义。

---

### 逐行解析：

| ord   | 矩阵范数 (norm for matrices)         | 向量范数 (norm for vectors)              |
| ----- | ------------------------------------ | ---------------------------------------- |
| None  | Frobenius 范数                       | 2-范数（欧几里得范数）                   |
| 'fro' | Frobenius 范数                       | 不适用（仅用于矩阵）                     |
| 'nuc' | 核范数（Nuclear norm）               | 不适用（仅用于矩阵）                     |
| inf   | 每行绝对值之和的最大值（即最大行和） | 最大绝对值（即 ∞-范数）                  |
| -inf  | 每行绝对值之和的最小值（即最小行和） | 最小绝对值（即 -∞-范数）                 |
| 0     | 不支持（无定义）                     | 非零元素个数（即 0-范数，衡量稀疏性）    |
| 1     | 每列绝对值之和的最大值（即最大列和） | 如下（见公式）                           |
| -1    | 每列绝对值之和的最小值（即最小列和） | 如下（见公式）                           |
| 2     | 2-范数（最大奇异值）                 | 如下（见公式）                           |
| -2    | 最小奇异值                           | 如下（见公式）                           |
| other | 不支持（仅适用于向量）               | 通用 p-范数：`sum(abs(x)**ord)**(1/ord)` |

---

### 详细解释：

#### 1. **Frobenius 范数**（`None` 或 `'fro'`）
- **矩阵**：相当于将矩阵“展平”成向量后的 2-范数。
  $$
  \|A\|_F = \sqrt{\sum_{i,j} |a_{ij}|^2}
  $$
- **向量**：等价于 2-范数（欧氏长度）。

> 注意：`'fro'` 只能用于矩阵。

---

#### 2. **核范数**（`'nuc'`）
- **矩阵**：所有奇异值的和。
  $$
  \|A\|_* = \sum \sigma_i
  $$
  这是矩阵的“迹范数”，常用于低秩矩阵恢复问题。
- **向量**：不适用。

---

#### 3. **∞-范数**（`inf`）
- **矩阵**：每行绝对值之和的最大值，也叫 **最大行和范数**。
  $$
  \|A\|_\infty = \max_i \sum_j |a_{ij}|
  $$
- **向量**：最大绝对值（即 ∞-范数）。
  $$
  \|x\|_\infty = \max_i |x_i|
  $$

---

#### 4. **-∞-范数**（`-inf`）
- **矩阵**：每行绝对值之和的最小值。
- **向量**：最小绝对值。

> 实际应用较少。

---

#### 5. **0-范数**（`0`）
- **矩阵**：不支持（因为没有标准定义）。
- **向量**：非零元素的个数。
  $$
  \|x\|_0 = \#\{i : x_i \neq 0\}
  $$
  常用于稀疏性度量。

---

#### 6. **1-范数**
- **矩阵**：每列绝对值之和的最大值（最大列和范数）。
  $$
  \|A\|_1 = \max_j \sum_i |a_{ij}|
  $$
- **向量**：绝对值之和（1-范数）。
  $$
  \|x\|_1 = \sum |x_i|
  $$

---

#### 7. **-1-范数**
- **矩阵**：每列绝对值之和的最小值。
- **向量**：同上（即 1-范数），但注意这里的 `-1` 并不表示负的 1-范数，而是特殊定义。

> 通常不常用。

---

#### 8. **2-范数**（`2`）
- **矩阵**：最大奇异值（谱范数）。
  $$
  \|A\|_2 = \sigma_{\max}(A)
  $$
  表示矩阵作为线性变换时的最大“拉伸倍数”。
- **向量**：欧几里得范数。
  $$
  \|x\|_2 = \sqrt{\sum x_i^2}
  $$

---

#### 9. **-2-范数**（`-2`）
- **矩阵**：最小奇异值。
  $$
  \|A\|_{-2} = \sigma_{\min}(A)
  $$
  与条件数有关。
- **向量**：同 2-范数。

---

#### 10. **其他 ord 值**（如 `p=3`, `p=4` 等）
- **矩阵**：不支持（只对向量有效）。
- **向量**：p-范数定义为：
  $$
  \|x\|_p = \left( \sum |x_i|^p \right)^{1/p}
  $$
  其中 `p` 是 `ord` 的值。

> 例如：
> - `ord=3`: $\|x\|_3 = (\sum |x_i|^3)^{1/3}$
> - `ord=-3`: $\|x\|_{-3} = (\sum |x_i|^{-3})^{-1/3}$，但需注意可能无意义或趋于无穷。

---

### 总结

| 类型            | 说明                      |
| ------------- | ----------------------- |
| **Frobenius** | 矩阵的“平方和开根号”，类似于向量的 2-范数 |
| **Nuclear**   | 所有奇异值之和，用于低秩优化          |
| **∞ / -∞**    | 最大/最小行和或最大/最小绝对值        |
| **0-范数**      | 非零元素个数（向量）              |
| **1-范数**      | 列和最大值（矩阵），绝对值和（向量）      |
| **2-范数**      | 最大奇异值（矩阵），欧氏长度（向量）      |
| **p-范数**      | 通用公式：$\left(\sum \lvert x_i \rvert ^p\right)^{1/p}$        |

---

### 应用场景示例：

- **机器学习**：使用 2-范数正则化（L2 正则）防止过拟合。
- **信号处理**：使用 1-范数促进稀疏解（L1 正则）。
- **图像压缩**：使用核范数进行低秩近似。
- **数值稳定性分析**：使用 2-范数估计矩阵的“大小”。

---

如果你是在使用 Python 的 `numpy.linalg.norm()` 函数，可以这样调用：

```python
import numpy as np

A = np.array([[1, 2], [3, 4]])
print(np.linalg.norm(A, ord='fro'))  # Frobenius norm
print(np.linalg.norm(A, ord=2))      # 2-norm (largest singular value)
print(np.linalg.norm(A, ord=1))      # max column sum
```
