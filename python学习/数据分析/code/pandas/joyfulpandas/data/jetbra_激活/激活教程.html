3. Windows 系统的 IDEA 激活

  • 先执行 uninstall-all-users.vbs，直接双击打开，此步骤是为了防止之前有过激活信息，确保当前环境变量下没有激活工具的变量信息。
  • 可先执行卸载脚本，再进行后面的激活操作，避免激活失败。

出现弹框 done 说明成功

然后再执行 install-current-user.vbs，直接双击打开即可

等待约 10 秒后出现 done 弹框，即为成功

然后重新打开 IDEA 应用，输入对应的激活码即可！

4. Mac 系统的 IDEA 激活

  1. 打开终端
  2. 运行 uninstall.sh，此步骤一键清除旧激活信息，防止激活失败

使用命令：sh 路径/uninstall.sh，注意空格

出现 done 说明成功

接着执行 install.sh，出现 done 和 kill xxx 提示即为成功

重新打开 IDEA，输入激活码。
