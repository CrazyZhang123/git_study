## 4、ssh免密登录
#### **同一把钥匙，挨台服务器复制公钥**

本地执行

```
ssh-keygen          # 一路回车，生成 ~/.ssh/id_rsa 和 id_rsa.pub 
ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.235.22`
```

如果你有多台服务器，比如：

- 192.168.235.22
    
- 192.168.235.23
    
- 210.28.133.13
    

可以直接多跑几次 `ssh-copy-id`：

```
ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.235.22 
ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.235.23 
ssh-copy-id -i ~/.ssh/id_rsa.pub root@210.28.133.13
```

原理是：  
每台服务器都有自己的 `~/.ssh/authorized_keys` 文件，`ssh-copy-id` 就是把你的公钥追加到那里面去。你把同一个公钥追加到 N 台服务器上，你这台客户端的这把私钥就能登上所有这些服务器。

之后登录就直接：

```
ssh root@192.168.235.22 
ssh root@192.168.235.23 
ssh root@210.28.133.13
```
