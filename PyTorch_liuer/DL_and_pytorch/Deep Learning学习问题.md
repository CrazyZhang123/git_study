

# 1、残差连接

**残差连接的核心思想**是==在网络的一层或多层之间引入直接连接，使得这些层的输出不仅包括经过非线性变换的特征，还包括未经处理的输入特征==。这样做的**目的**是==允许网络学习到的是输入和输出之间的残差（即差异），而不是直接学习一个完整的映射==。这种方式有助于梯度在训练过程中更有效地回流，减轻深度网络中梯度消失的问题。

[[残差连接]]

| 方法名称                              | 核心定义                                                     | 关键操作步骤                                                 |
| ------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| Mean（简单平均法）                    | 直接对所有模型的对应参数取算术平均，不做任何过滤或调整。     | 仅一步：参数值求和 ÷ 模型数量（如模型A参数=2、模型B参数=-1，平均后=0.5） |
| Trim + Dis. Mean（修剪+不重叠均值法） | 先剔除冗余（不重要）参数，再对剩余“非冗余参数”按“不重叠规则”取均值（仅融合对任务关键的参数）。 | 两步：1. 修剪：移除所有模型中“不重要的参数”（如幅度最小的20%）；2. 均值：对剩余参数取平均（不调整符号） |
| TIES（完整抗干扰法）                  | 先修剪冗余参数，再解决符号冲突，最后对“无冗余、无冲突”的参数取均值，是前两者的进阶优化。 | 三步：1. 修剪：移除冗余参数；2. 符号筛选：统一参数符号（如少数服从多数）；3. 均值：对处理后参数取平均 |
